\# Login Test - Android



Aplicación básica de inicio de sesión desarrollada en Android Studio con Java.



\## Características



\- Pantalla de login

\- Validación de usuario y contraseña

\- Interfaz simple con ConstraintLayout

\- Uso de Toast para mensajes



\## Tecnologías usadas



\- Java

\- Android Studio

\- XML (Layouts)

\- ConstraintLayout

\## Estructura del proyecto



\- `Login.java` → Lógica del login

\- `activity\_login.xml` → Diseño de la interfaz



\## Cómo ejecutar



1\. Clonar el repositorio

2\. Abrir en Android Studio

3\. Ejecutar en emulador o dispositivo físico



\## Notas



Este proyecto es de práctica para entender la estructura básica de un login en Android.
